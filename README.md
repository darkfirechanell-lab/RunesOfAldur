RunesOfAldur is a plugin for ExileCore2 (Core 2) designed for Path of Exile 2's Runes of Aldur league mechanic.

When you open the Ezomyte Remnant altar, the plugin automatically reads the available reward options displayed on screen and cross-references them in real-time against poe.ninja prices (fetched automatically for your current league).

Features:

Draws a green border around the most valuable reward option
Displays the price in Exalted or Divine Orbs next to each row
Prices update every 30 minutes automatically
Fully configurable — border thickness, price display toggle, price position slider
No dependency on other plugins — works standalone
Created with the assistance of Claude (Anthropic AI).
